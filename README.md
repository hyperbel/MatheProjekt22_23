# MatheProjekt22_23
Mathe Abschluss schulprojekt in Python, zur Berechnung von Mathe Formeln 

## Programm benutzen
### Windows
* Repo runterladen
* Ordner mit einer IDE öffnen 
* Main.py ausführen
* 
### Linux
* `git clone https://github.com/hyperbel/MatheProjekt22_23`
* `cd MatheProjekt22_23`
*  `pip install requirements.txt`
*  `python3 main.py`
